smugapi by clalancette

Pythonic API for accessing SmugMug

SmugAPI is a python module for accessing SmugMug data. It has API's for uploading, downloading, and manipulating SmugMug pictures, albums, and categories, all in a convenient python object oriented way. It also includes a command-line utility (smugtool) to perform common operations.

Dependencies

python-pycurl

Install

On an RPM based distro, building a smugapi RPM is a simple matter of running "make rpm", and then installing the resulting RPM. For other users, you should be able to put smugapi.py wherever your python site-packages is located (typically something like /usr/lib/python2.6/site-packages), and then put smugtool in /usr/bin. If you don't have administrator rights on your machine (and hence can't write files to those directories), you can just use smugtool and smugapi in-place.
